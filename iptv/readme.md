## IPTV with VCL Media Player

After reading 
[Elorm Daniel's post on VLC](https://x.com/elormkdaniel/status/2015511743309689250)
, I decided to document my experience with IPTV and VLC media player.

Set a Network Stream for 

    Media -> Open Network Stream -> (enter a .m3u formatted URL) -> Play

    or

    Ctrl+N

See the Playlist of channels

    View -> Playlist

    or

    Ctrl+L

Chromecast Support

    Playback -> Renderer
